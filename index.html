Welcome to CrystalBall Pages.

CrystalBall is tool for Activiti Business Process Management Platform. CrystalBall allows use of simulation for:

  • decision support - for production workflow (e.g. should we add now more resources to the system to meet due dates?)
  • optimisation and understanding - testing changes and understanding their impacts.
  • training - simulation can be used to train staff before roll out.
  • ...

What CrystalBall provides

CrystallBall provides the following tools:
  • reports to monitor activiti process engine(available reports)
  • simulation engine

CrystalBall is unique

No need to:
  • create separate simulation models and engines
  • create different reporting for simulation
  • feed simulation engine with a lot of data

CrystalBall simulator is based workflow engine used in the solution. That's why it is easy to copy data and start simulation and be able to replay workflow behavior from the history

Examples

Reports
  • Draw into process diagram
  • Find process bottleneck
  • What is the percentage of after due date tasks?
  • Audit trail diagram
Simulation
  • Use simulation to optimize process bottleneck
  • Use process playback in simulation and optimization.
